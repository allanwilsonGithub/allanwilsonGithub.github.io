RULES:
A) Object is PARTATIIVI when...
1. Lause on negative = Etkö halua omenaa? En halua omenaa.
2. Objecti is ainesana = Haluatko kahvia, minä juon vettä, voitko keittää (brew) kahvia?
3. Lause on prosessi = Marja rakastaa Pekkaa, luetko viela sitä samaa kirjaa

B) Objekti on AKKUSATIIVI when...
1. Object is singular = Hän avaa ikkunan, vien kirjeen postiin. Millaisen auton?

C) Object is NOMINATIVE when...
1. Object is plural = Vien kirjeet postiin
2. Number of something (the something is Partative) = kaksi voileipaa
